Basics of Python

About Python:

  1. Python is one of many coding languages like HTML
  2. Some of Python's main functions include:
  3. Print function, which displays an output
  4. Type function, which shows the type of object a particular value is
  5. Input function, which both displays an output and returns an output based on the user's input
  6. Python also has other cool features like if statements and while loops
